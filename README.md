# keshav-jha.dev

Personal portfolio website of Keshav Jha — software engineer at Oracle (MySQL InnoDB), builder of storage engines and distributed systems.

Built with Next.js 14 (App Router) + TypeScript, styled with Tailwind CSS, deployed on Vercel.

## stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + CSS Variables
- **Theme:** next-themes (dark / light)
- **Blog:** Medium RSS feed (fetched via API route)
- **Contact:** mailto link
- **Deployment:** Github

## running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
